# Instrucciones rápidas (Tarea HTML sin CSS)

1) Reemplace los archivos multimedia:
   - imagenes/foto1.jpg → ponga su foto.
   - videos/presentacion.mp4 → vídeo propio de ≤60 s.
   - audios/cancion.mp3 → un clip/música local (permiso propio).
2) Actualice nombres reales, listas y la tabla en cada biografía.
3) Edite el ID del video de YouTube en cada archivo HTML (busque 'youtube_id').
4) Abra index.html para probar los enlaces.

## Publicación gratuita (opciones)
- GitHub Pages: suba esta carpeta como repositorio y habilite Pages.
- Netlify Drop: arrastre la carpeta al sitio de Netlify Drop.
- Vercel: importe el repo y despliegue como 'Static'.
(En todas, mantenga la estructura de carpetas igual.)

## Entrega en Moodle
- Genere un PDF con el código fuente y capturas del sitio.
  Sugerencia: Abra cada HTML en su editor y use 'Imprimir → Guardar como PDF'.
- Pegue el enlace al sitio publicado en los comentarios del envío.
